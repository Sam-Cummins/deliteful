Module: deliteful/Store

deliteful/Store

require(["deliteful/Store"], function (Store) {
  new Store();
});

Custom element to create an instance of a memory store object. It is particularly useful in markup, when creating store programatically, it's easier to just create a store instance directly rather than using the custom element. Note that it is only parsed once at attachment time. Further modifications to the store must be made through the store API instead of the DOM API.

Source:

Example
<body>
  <d-store>
   { "label": "France", "sales": 500, "profit": 50, "region": "EU" },
   { "label": "Germany", "sales": 450, "profit": 48, "region": "EU" },
   { "label": "UK", "sales": 700, "profit": 60, "region": "EU" }
  </d-dstore>
</body>
