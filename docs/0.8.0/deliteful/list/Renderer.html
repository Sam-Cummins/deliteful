deliteful/list/Renderer

require(["deliteful/list/Renderer"], function (Renderer) {
  new Renderer();
});

The base class for a widget that render an item or its category inside a deliteful/list/List widget.

This base class provide all the infrastructure that a deliteful/list/List widget expect from a renderer, including keyboard navigation support.

Focusability and Keyboard navigation order for a renderer instance is defined using the navindex attribute on the rendered nodes:

  • no navindex attribute value means that the node is not focusable
  • a navindex attribute means that the node is focusable. When navigating the renderer instance using arrow keys, the traversal order is the following:
    • the nodes with the lowest navindex value comes first
    • if two nodes have the same navindex value, the one that is before the other one in the DOM comes first.

updateFocusableChildren()

This method update the list of children of the renderer that can be focused during keyboard navigation. If the list of navigable children of the renderer is updated after the render step has been executed, this method must be called to take into account the new list. If the list of navigable children is defined during the render step, there is no need to call this method.
