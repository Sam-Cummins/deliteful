Module: deliteful/ProgressIndicator

deliteful/ProgressIndicator

require(["deliteful/ProgressIndicator"], function (ProgressIndicator) {
  new ProgressIndicator();
});

A widget that displays a round spinning graphical representation that indicates that a task is ongoing.

This widget starts hidden and the spinning animation starts when the widget becomes visible. Default widget size is 40x40px.

Source:
  • ProgressIndicator.js, line 10
Examples

Set the "active" property to true to make the widget visible when it starts.

<d-progress-indicator active="true"></d-progress-indicator>

Use style properties "width" and "height" to customize the widget size

<d-progress-indicator active="true" style="width: 100%; height: 100%"></d-progress-indicator>

active :boolean

Set to false to hide the widget and stop any ongoing animation. Set to true to show the widget: animation automatically starts unless you set a number to the "value" property.

speed :string

The relative speed of the spinning animation. Accepted values are "slow", "normal" and "fast". Other values are converted to "normal". Note that the actual/real speed of the animation depends of the device/os/browser capabilities.

Type:
  • string
Default Value:
  • "normal"
Source:

value :number

A value from 0 to 100 that indicates a percentage of progression of an ongoing task. Set the value to NaN to hide the number and start the spinning animation. Negative values are converted to 0 and values over 100 are converted to 100.

Type:
  • number
Default Value:
  • NaN
